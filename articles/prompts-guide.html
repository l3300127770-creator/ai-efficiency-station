一、办公效率提示词

1.1 周报生成

你是一个专业的职场写作助手。请帮我写一份本周工作周报。
本周完成的工作：【填写你的工作内容】
下周计划：【填写下周计划】
要求：简洁专业，重点突出成果，500字以内。

1.2 商务邮件

帮我写一封商务邮件，收件人是【职位/关系】，主题是【邮件主题】。
核心内容：【要表达的关键信息】
语气要求：正式/友好/委婉（选一个）

1.3 会议纪要

请根据以下会议内容，整理一份结构化会议纪要，包含：议题、讨论要点、决议事项、责任人和截止时间。
会议内容：【粘贴会议记录或要点】

1.4 项目方案

帮我写一份项目方案。项目名称：【名称】，目标：【目标】，预算：【预算范围】，周期：【时间】。
要求包含：背景分析、执行计划、预期成果、风险预案。

二、自媒体写作提示词

2.1 公众号文章

你是一个资深公众号写手。请写一篇关于【主题】的文章。
目标读者：【描述受众】
字数：800-1500字
风格：轻松有料，有案例，有金句
结构：吸引人的开头 + 3-5个要点 + 总结行动建议

2.2 小红书文案

帮我写一篇小红书笔记，主题是【主题】。
要求：标题带emoji，正文分段，每段简短有力，结尾带互动引导。
参考风格：真诚分享、干货满满、适合收藏。

2.3 知乎回答

请回答知乎问题：【问题内容】
要求：先给结论，再展开论述，有数据或案例支撑，1000字左右。
风格：专业但不枯燥，有个人见解。

2.4 爆款标题

请帮我生成10个爆款标题，主题是【主题】。
要求：带数字、带悬念、适合【平台名称】的风格。
参考结构：数字+痛点+解决方案 / 悬念+反转 / 对比+冲突

三、短视频提示词

3.1 视频脚本

帮我写一个60秒短视频脚本，主题是【主题】。
平台：抖音/快手/B站（选一个）
结构：前3秒钩子 + 中间内容 + 结尾引导关注
风格：口语化，节奏快，有反转或亮点

3.2 分镜脚本

请根据以下视频脚本，生成分镜脚本。
脚本内容：【粘贴脚本】
输出格式：镜头编号 | 画面描述 | 台词/旁白 | 时长 | 备注

3.3 字幕与配音

请将以下内容改写为适合AI配音的口语化脚本，要求：短句为主，语气自然，适合【平台】风格。
原始内容：【粘贴文字】

四、电商与客服提示词

4.1 产品描述

请帮我写一段产品描述文案。
产品名称：【产品名】
核心卖点：【列出2-3个卖点】
目标客户：【描述目标人群】
要求：突出痛点和解决方案，200字以内。

4.2 客服回复

你是一个专业客服。客户说：【客户的问题或投诉】
请用友好、专业的语气回复，解决问题的同时维护品牌形象。
要求：先共情，再给方案，最后确认。

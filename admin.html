📧 邮件订阅配置（Mailchimp）

用户填邮箱后自动收到资源邮件，全自动运行，零维护。

第一步：注册Mailchimp（免费）

1️⃣

注册账号

访问 mailchimp.com → 点 Sign Up Free → 填写邮箱和密码
免费版支持：500联系人 + 每月1000封邮件 + 自动化工作流

第二步：创建Audience（受众）

2️⃣

设置受众列表

登录后 → Audience → All contacts → Create Audience
Audience name: AI效率站订阅者
Default from email: 你的真实邮箱
Default from name: AI效率站

第三步：创建Signup Form（注册表单）

3️⃣

获取表单URL

Audience → Signup forms → Form builder
或者直接用 Embedded forms：
Audience → Signup forms → Embedded forms
复制表单中的 action="https://xxx.usXX.list-manage.com/subscribe/post?u=XXXXX&id=XXXXX"
这个URL就是你需要填入网站的 https://wwazk.lanzouq.com/imCva3sfaxmh

第四步：替换网站中的占位URL

4️⃣

全局替换

在编辑器中按 Ctrl+H → 搜索 https://wwazk.lanzouq.com/imCva3sfaxmh → 替换为你的Mailchimp表单URL
同时找到 name="b_XXXXX" 中的 XXXXX 替换为你的真实值（防机器人字段）
涉及文件：index.html、3篇文章、shop.html（共5个文件）

第五步：创建Welcome Email（欢迎邮件）

5️⃣

设置自动化欢迎邮件

Mailchimp → Automations → Create → Welcome new subscribers

第六步：创建Automation（自动化推送）

6️⃣

新文章自动推送（可选）

Mailchimp → Automations → Create → 自定义
触发条件：当Audience有新联系人时 → 发送最新文章摘要
或者：定期（每周/每月）发送Newsletter汇总

💡 替代方案：Resend（更技术向）

如果你熟悉API调用，可以用 Resend（免费3000封/月）。
需要配合Cloudflare Workers做后端API，但自定义程度更高。
适合想完全控制邮件内容和发送逻辑的用户。

⚠️ 注意事项

1. Mailchimp免费版有品牌Logo在邮件底部（可接受）
2. 确保邮件内容不被判定为垃圾邮件（避免过多营销词汇）
3. 资源下载链接建议用网盘分享，不要直接附件（避免被拦截）
4. 首次发送前用自己的邮箱测试一遍

🚀 部署上线指南

💡 推荐方案：Cloudflare Pages（免费+国内可访问）

1. 注册 Cloudflare Pages
2. 创建项目，上传整个网站文件夹
3. 绑定你的域名（在Cloudflare DNS设置）
4. 自动HTTPS，全球CDN，免费额度足够
5. 后续更新：直接上传新文件即可

💡 备选方案：GitHub Pages

1. 创建GitHub仓库，上传所有文件
2. 在 Settings → Pages 启用
3. 绑定自定义域名
4. 每次 git push 自动更新

⚠️ 部署前检查

确保所有占位内容（邮箱、链接、收款码）已替换为真实内容。可以用本页面的检查清单逐项确认。

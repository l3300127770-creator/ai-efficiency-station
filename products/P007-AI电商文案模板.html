🏷️ 商品标题优化模板
好标题=高点击率=更多流量=更多销量。掌握标题公式，让AI帮你批量生成

🏷️ 淘宝/天猫标题公式

核心技能
标题万能公式
📌 淘宝标题万能公式（30字以内）： 品牌词 + 核心关键词 + 属性词 + 场景词 + 修饰词 示例拆解： 产品：女士防晒霜 ✅ "XX品牌 防晒霜女 面部隔离 SPF50+ 清爽不油腻 夏季户外军训专用" 各词库说明： • 品牌词：你的品牌名（有品牌必加） • 核心关键词：用户搜索最多的词（用淘宝搜索下拉词找） • 属性词：规格/材质/颜色/功效（如"SPF50+""清爽""持久"） • 场景词：使用场景（如"夏季""户外""军训""通勤"） • 修饰词：促销/情感词（如"2025新款""网红同款""买一送一"） 🔍 找关键词方法： 1. 打开淘宝搜索框，输入品类词，看下拉推荐 2. 看竞品标题用了什么词 3. 生意参谋-搜索分析（付费） 4. 免费工具：阿里指数、淘宝直通车关键词工具
AI提示词
你是一个淘宝SEO优化专家。请帮我优化以下商品的标题。 商品信息： - 产品名称：[填写] - 品类：[填写] - 核心卖点：[填写2-3个] - 目标人群：[填写] - 价格区间：[填写] - 使用场景：[填写] 要求： 1. 生成5个不同版本的标题（每个30字以内） 2. 标题必须包含：品牌词+核心关键词+属性词+场景词 3. 关键词要符合用户搜索习惯（用淘宝下拉词风格） 4. 不要堆砌无关关键词 5. 每个标题标注侧重方向（如：侧重搜索/侧重点击/侧重转化） 6. 最后推荐一个最佳标题并说明理由

📱 拼多多/抖店标题技巧

多平台
平台差异化写法
📌 拼多多标题特点： - 标题可以更长（最多60字） - 用户更看重性价比，标题要突出"便宜""实惠" - 可以加促销信息："买一送一""第二件半价" - 示例："【买一送一】防晒霜女面部SPF50+清爽不油腻学生党平价好用夏季防紫外线" 📌 抖音小店标题特点： - 标题要短、直击痛点 - 用户是被视频种草后来搜索的，标题要承接视频内容 - 突出"网红""同款""爆款"等社交证明词 - 示例："网红同款防晒霜 SPF50+清爽不假白 夏天必备"
⭐ 五点描述/卖点提炼模板
用FABE法则提炼卖点，让顾客看了就想买

🎯 FABE卖点提炼法

转化神器
FABE法则模板
📌 FABE法则 = Feature(特征) + Advantage(优势) + Benefit(利益) + Evidence(证据) 万能卖点模板： 卖点1： F（特征）：[产品有什么特点/成分/技术] A（优势）：[相比竞品有什么优势] B（利益）：[能给用户带来什么好处] E（证据）：[有什么证据支撑（数据/认证/评价）] 示例 - 防晒霜卖点： 卖点1 - 防晒力： F：SPF50+ PA++++高倍防晒 A：比普通防晒霜防护力高3倍 B：户外暴晒一天也不会晒黑晒伤 E：通过国家防晒特证认证，10万+好评 卖点2 - 肤感： F：水润质地，添加玻尿酸保湿成分 A：上脸不油腻不假白，成膜速度快 B：化妆不搓泥，油皮也能放心用 E：小红书5000+博主实测推荐 卖点3 - 性价比： F：60ml大容量，可用2-3个月 A：日均成本不到0.5元 B：学生党也能无压力入手 E：天猫防晒霜销量TOP3
AI提示词
你是一个电商文案专家，擅长用FABE法则提炼产品卖点。请帮我为以下产品提炼5个核心卖点。 产品信息： - 产品名称：[填写] - 产品类型：[填写] - 主要成分/材质/技术：[填写] - 价格：[填写] - 目标人群：[填写] - 竞品差异点：[填写] 要求： 1. 每个卖点用FABE法则：特征→优势→利益→证据 2. 卖点按重要性排序（最重要的排第一） 3. 用户利益要具体可感知（不要笼统说"好用"） 4. 证据要可信（数据/认证/销量/评价） 5. 语言简洁有力，每条控制在3-4行 6. 最后给出一个适合放在标题里的核心卖点（10字以内）

📄 详情页黄金结构

高转化
万能详情页结构
📌 详情页黄金5步结构： 第一步：痛点场景（让顾客觉得"说的就是我"） "夏天出门5分钟，脸上就开始泛油光？" "涂了防晒还是晒黑了？可能是你选错了产品" → 用1-2个具体场景描述用户的痛点 第二步：解决方案（引出你的产品） "你需要的是一款[核心卖点]的[产品名]" → 简洁引出产品，给出解决方案 第三步：卖点证明（用事实说话） 卖点1：[核心卖点] + [数据/对比/认证] 卖点2：[第二卖点] + [使用场景/效果] 卖点3：[第三卖点] + [用户评价/销量] → 每个卖点配一张效果图或对比图 第四步：信任保障（消除顾虑） ✅ [资质认证/品牌背书] ✅ [售后保障/退换政策] ✅ [销量数据/好评率] ✅ [真人使用对比/买家秀] 第五步：行动号召（推动下单） "限时[X]折，到手仅¥[价格]" "前100名下单送[赠品]" "点击立即购买，今天下单今天发货" → 制造紧迫感，推动立即行动
AI提示词
你是一个电商详情页文案专家。请帮我写一个商品详情页文案。 商品信息： - 产品名称：[填写] - 价格：[填写] - 核心卖点：[填写3个] - 目标人群：[填写] - 主要竞品及差异：[填写] - 可用的信任背书：[填写认证/销量/评价等] 要求： 1. 用"痛点→方案→证据→保障→行动"黄金5步结构 2. 痛点部分要具体（描述2个使用场景） 3. 卖点部分每个都要有数据或证据支撑 4. 信任保障部分列出4个以上信任元素 5. 行动号召要制造紧迫感（限时/限量/赠品） 6. 语言简洁有力，每段不超过3行 7. 标注每个板块建议配什么类型的图片

🔍 一、如何用AI找热门选题

很多人做短视频的第一步就卡住了——不知道拍什么。其实，AI可以帮你解决这个问题。以下是具体操作步骤：

  1. 打开ChatGPT或Kimi（推荐Kimi，免费且支持中文联网搜索）。在对话框中输入以下提示词：
    你是一个短视频选题专家。请帮我找出当前抖音/小红书上最近一周的10个热门选题，要求： 1. 适合个人创作者（不需要团队配合） 2. 制作门槛低（不需要专业设备） 3. 变现潜力高（能带货或接广告） 4. 按热度从高到低排列，每个选题附上推荐理由
  2. 筛选AI给出的选题。AI会给你一个列表，但不要直接用，需要做第二轮筛选。继续输入：
    针对上面第 [1] 个选题，请帮我分析： 1. 这个选题的目标人群是谁？ 2. 目前抖音上做这个选题的头部账号有哪几个？他们的平均播放量是多少？ 3. 这个选题适合做口播、vlog还是图文混剪？ 4. 给我3个切入角度，要和现有账号有差异化
  3. 交叉验证选题热度。打开抖音/小红书，搜索你选中的关键词，看前10条视频的点赞量。如果大部分视频点赞过千，说明选题热度OK。如果最高点赞才几百，说明这个选题可能已经过气或受众太窄。

📊 二、选题评估方法：三个维度打分

不是所有热门选题都适合你。用这个"三维评估法"给选题打分，每个维度1-5分，总分超过10分的选题才值得做：

评估维度评分标准加分项
热度搜索量、相关视频播放量近期有上升趋势 +2分
竞争度头部账号数量、内容同质化程度有差异化空间 +2分
变现潜力能否带货、接广告、引流私域客单价>50元 +2分
💡
小白建议：前期不要追求完美选题，先做起来。选一个你感兴趣、能持续产出内容的方向，比纠结哪个选题最火更重要。

✍️ 三、AI生成脚本的完整流程

选好题之后，接下来就是写脚本。一个好的短视频脚本需要包含：开头钩子、正文内容、结尾引导。用AI生成脚本的标准流程如下：

  1. 确定视频类型和时长。先告诉AI你要做什么类型的视频。口播类建议60-90秒，混剪类建议30-60秒，vlog类建议2-3分钟。
  2. 输入脚本生成提示词：
    你是一个短视频脚本创作专家。请帮我写一条 [选题内容] 的短视频脚本，要求： 1. 视频类型：[口播/混剪/vlog] 2. 视频时长：[60秒] 3. 目标平台：[抖音/小红书] 4. 目标人群：[描述你的目标观众] 脚本格式要求： - 开头3秒要有强力钩子（提问/冲突/反常识） - 正文分3个要点，每个要点配一个具体案例 - 结尾要有行动号召（关注/点赞/评论引导） - 标注每段的预估时长 - 标注需要配什么画面/素材
  3. 优化和调整脚本。AI生成的第一版通常偏长或语气偏书面。继续输入：
    请对上面的脚本做以下优化： 1. 把所有书面语改成口语化表达，像朋友聊天一样 2. 缩短到 [60秒] 以内（按每秒4个字计算） 3. 开头钩子再强化，要让人停下来看 4. 每个要点加一句金句/总结，方便截图传播
  4. 审读并微调。拿到最终脚本后，自己读一遍，确保语句通顺、逻辑清晰。把拗口的地方改成自己的说话习惯。

📋 四、5个实用脚本模板

以下5种脚本模板覆盖了90%的短视频类型，直接套用即可：

模板1：知识科普型（60秒口播）

【开头钩子 3秒】你知道 [反常识/惊讶事实] 吗？ 【问题展开 10秒】很多人以为 [常见误区]，但其实 [真相]。 【要点1 15秒】第一，[知识点]。举个例子，[具体案例]。 【要点2 15秒】第二，[知识点]。比如 [具体案例]。 【要点3 10秒】第三，[知识点]。[简短案例]。 【总结 5秒】记住这三点：[复述要点]。 【引导 2秒】觉得有用就点赞收藏，关注我学更多！

模板2：产品种草型（45秒）

【开头 3秒】被种草了！[产品] 也太好用了吧！ 【痛点 8秒】之前我一直被 [问题] 困扰，试过 [其他方案] 都不行。 【使用体验 20秒】用了 [产品] 之后，[具体效果1]，[具体效果2]，最让我惊喜的是 [亮点]。 【使用方法 8秒】用法很简单，[简述步骤]。 【引导 6秒】链接放购物车了，想试试的姐妹扣1！

模板3：故事/经历型（90秒）

【开头钩子 5秒】[时间]前，我还 [糟糕的状况]，现在 [好的结果]。 【困境 20秒】那时候 [详细描述困难]，每天都很 [情绪]。 【转折 15秒】直到有一天，[发生了什么/发现了什么]。 【行动 25秒】我开始 [具体做了什么]，第一步 [行动1]，然后 [行动2]。 【结果 15秒】[时间]后，[具体成果]，最重要的是 [感悟]。 【鼓励 10秒】所以如果你现在也 [类似处境]，别放弃。[鼓励的话]。关注我，分享更多 [领域] 干货。

模板4：对比测评型（60秒）

【开头 3秒】[A] 和 [B] 到底哪个好？今天实测给你看！ 【外观对比 10秒】先看外观，[描述A]，[描述B]，我个人更喜欢 [选择]。 【功能对比 20秒】功能上，[A的优势]、[B的优势]，但在 [关键场景] 上 [哪个更好]。 【价格对比 10秒】价格方面，A卖 [价格]，B卖 [价格]，性价比 [结论]。 【总结 12秒】总的来说，如果你 [场景1] 选A，如果你 [场景2] 选B。 【互动 5秒】你会选哪个？评论区告诉我！

模板5：干货清单型（60秒）

【开头 3秒】[领域] 必备的 [数字] 个 [工具/技巧/网站]，建议收藏！ 【第1个 10秒】第一个，[名称]。它能 [功能]，我用了 [时长]，[效果]。 【第2个 10秒】第二个，[名称]。特别适合 [场景]，[亮点]。 【第3个 10秒】第三个，[名称]。最大的优势是 [优势]。 【第4个 10秒】第四个，[名称]。[功能+亮点]。 【第5个 10秒】第五个，[名称]。[功能+亮点]。 【总结 5秒】以上5个全部免费，链接都放在评论区了！ 【引导 2秒】关注我，每天分享实用干货！

⚠️ 五、常见坑和避坑指南

🚨
坑1：直接用AI生成的内容不做修改。AI生成的内容有明显的"AI味"，观众能感觉到。一定要用自己的话重新说一遍，加入个人经历和观点。
🚨
坑2：选题太宽泛。"教你学英语"这种题目太大，没人看。改成"每天5分钟，一个月记住3000个单词"就具体多了。AI也可以帮你把大选题缩小：输入"请帮我把这个选题缩小到一个具体的痛点"。
🚨
坑3：脚本太长。60秒视频的脚本大约240字（每秒4个字），90秒大约360字。写多了念不完，写少了太空洞。让AI按字数精确控制。
🚨
坑4：忽略开头前3秒。短视频的生死就在前3秒。如果前3秒没有抓住注意力，用户直接划走。每个脚本的开头至少让AI给你3个版本，选最抓人的那个。
✅
避坑总结：AI是你的助手，不是替代者。用AI提高效率，但最终内容要有你自己的风格和温度。前期可以大量用AI辅助，逐步形成自己的创作模式后，AI的比重可以降低到50%左右。

📝 三、字幕自动生成方法

字幕是短视频的标配，用户很多时候是静音观看的。以下是生成字幕的两种最简单方法：

  1. 方法一：剪映自动识别（推荐）。导入音频/视频后，点击"文字" → "智能字幕" → "识别字幕"。剪映会自动识别语音内容并生成字幕，准确率在95%以上。生成后逐句检查，重点校对专有名词、数字、多音字。
  2. 方法二：先生成SRT字幕文件再导入。如果用的是外部配音（如微软TTS生成的音频），可以先用"通义听悟"或"飞书妙记"上传音频，自动转写成SRT字幕文件，再导入剪映。这种方式的好处是字幕文件可复用。
  3. 字幕样式设置。推荐设置：字体选"思源黑体"或系统默认黑体，字号适中（不要太大也不要太小），白色字体+黑色描边（2-3px），居中对齐。避免用花哨字体，影响阅读。

🎬 四、音画同步技巧

音画不同步是新手最常犯的错误之一，观众会立刻觉得"不专业"。以下是确保同步的关键技巧：

  1. 先配音，后配画面。正确顺序是：脚本 → 配音 → 字幕 → 画面。以音频为时间轴基准，画面去配合音频。
  2. 在剪映中利用波形对齐。音频轨道会显示声波波形，根据波形的高低起伏来判断每句话的开始和结束位置，把对应画面的出入点对齐到这些位置。
  3. 留出呼吸间隙。两句话之间留0.3-0.5秒的空隙，不要剪得太紧凑。这样观感更自然，也给观众消化信息的时间。
  4. 关键信息和画面同步。当配音说到"第一点""第二点"时，画面上要同时出现对应的文字或图标。这种音画同步会让视频看起来非常专业。

🎛️ 五、音频后期处理基础

AI生成的配音通常已经很干净了，但你可能还需要做一些简单的后期处理：

  1. 音量标准化。确保配音音量在-6dB到-3dB之间，不要太大（会爆音）也不要太小（听不清）。在剪映中选中音频轨道，通过音量滑块调节。
  2. 降噪处理。如果配音中有轻微底噪，在剪映中选中音频 → 点击"降噪" → 拉到50%-70%即可。不要拉满，否则声音会变得很闷。
  3. 添加背景音乐。纯配音会显得干巴巴的。在剪映"音频" → "音乐"中选一首轻快的背景音乐，音量调到配音的15%-25%即可。推荐使用剪映自带的"商用音乐"，避免版权问题。
  4. 音效点缀。在关键转折点可以加一些音效，比如"叮"的提示音、转场音效等。剪映的"音效"库里有很多免费音效可以直接用。
✅
实操建议：先花10分钟用AI生成配音，再花5分钟校对字幕，最后花5分钟调整音画同步。整个音频环节控制在20分钟以内。熟练之后10分钟就能搞定。

✂️ 三、剪映AI剪辑功能详解

剪映（CapCut）是目前国内最强大的免费视频剪辑工具，它内置了大量AI功能，能极大提高剪辑效率：

  1. 图文成片。把脚本文案粘贴进去，剪映会自动匹配素材、配音、字幕、背景音乐，一键生成视频。适合快速出片，但画面可能不够个性化。路径：打开剪映 → 图文成片 → 粘贴文字 → 选择风格 → 生成。
  2. 智能匹配素材。剪映的素材库非常丰富，在时间线上输入关键词，AI会自动推荐相关的视频片段和图片素材，直接拖到时间线就能用。比自己找素材快10倍。
  3. AI去水印/抠像。如果你使用了网上找的素材，可以用"智能抠像"功能去掉背景，或者用"消除笔"去掉水印。操作：选中素材 → 画面 → 智能抠像。
  4. 一键三连：调节+滤镜+美颜。如果有人出镜，剪映可以一键美颜、瘦脸、大眼。路径：选中视频 → 美颜美体 → 按需调节。不要调太夸张，自然最好。
  5. 智能字幕样式。剪映支持一键给字幕加花字效果（高亮、弹出、渐显等），让视频更有表现力。路径：选中字幕 → 花字 → 选择模板。

🌈 四、转场、特效、滤镜使用

好的转场和特效能提升视频质感，但切记：少即是多。

  1. 转场推荐。最常用的转场有3种：硬切（直接切换，最干净自然）、叠化（两段画面渐变过渡，适合情绪转换）、缩放（放大或缩小进入下一个画面，适合强调重点）。避免使用花里胡哨的转场（如翻转、旋转），会让视频显得廉价。
  2. 特效使用。特效主要用来强调重点。比如讲到关键数据时，可以加一个"放大镜"特效把数字放大；讲到重要观点时，加"闪烁"或"光效"吸引注意力。特效使用频率建议：60秒视频最多用2-3个特效。
  3. 滤镜选择。整体色调要统一，全片只用一个滤镜。推荐使用剪映的"清新""暖色""电影"系列滤镜，透明度调到50%-70%，不要100%满上。

🎬 五、完整剪辑流程演示

以下是用剪映制作一条60秒知识科普短视频的完整流程：

  1. 新建项目。打开剪映，点击"开始创作"，选择画布比例为9:16（竖屏），这是抖音/快手/小红书的标准比例。
  2. 导入素材。把你用AI生成的图片、视频片段全部导入到素材库。同时导入AI配音的音频文件。
  3. 放置音频。先把配音音频拖到音频轨道上，这是整条视频的时间基准。播放一遍确认音频完整无误。
  4. 添加字幕。使用"智能字幕" → "识别字幕"，自动为配音生成字幕。逐句检查校对，特别是专有名词和数字。然后给字幕选择一个统一样式（推荐白字黑描边）。
  5. 配画面。根据脚本标注的画面要求，把对应的图片/视频素材拖到视频轨道上，调整时长使其和音频对齐。每段画面之间可以加一个"叠化"转场。关键信息处用"文字"功能添加标题和要点。
  6. 添加背景音乐。在"音频" → "音乐"中选择一首轻快的背景音乐。音量调到15%-20%，确保不抢配音的风头。
  7. 加封面。在视频开头单独做一帧封面画面，放上标题文字。这帧封面会自动成为平台展示的缩略图。也可以单独上传一张精心设计的封面图。
  8. 导出视频。点击右上角"导出"，分辨率选1080P，帧率选30fps，格式选MP4。等待导出完成即可。
✅
时间参考：以上流程，新手第一次大约需要1-2小时。熟练之后可以控制在20-30分钟。使用"图文成片"功能可以把时间压缩到10分钟以内，但画面个性化程度会降低。
🚨
常见错误：不要在一段视频里混用太多风格（比如一半AI画面一半实拍），会让观众觉得割裂。选定一种风格后从头到尾保持一致。

📱 一、各平台算法解读

📝 三、标题和封面优化

标题和封面决定了用户是否点击。花在标题和封面上的时间，应该和做内容一样多。

标题优化5个公式：

  1. 数字+结果型："3个方法，让你的英语口语提升200%" — 数字具体，结果明确，用户一看就知道能得到什么。
  2. 提问型："为什么你存不下钱？99%的人都犯了这3个错" — 引发好奇，用户想知道自己是不是那99%。
  3. 对比型："月薪3千和月薪3万的人，每天下班后做的一件事不同" — 制造反差，激发点击欲。
  4. 指令型："别再用XX了！这个方法好用10倍" — 直接告诉用户该怎么做，适合干货类内容。
  5. 故事型："从负债10万到年入百万，我只做了这一件事" — 故事天然有吸引力，让人想看结局。

封面设计3个原则：

  1. 文字要大。封面在手机上显示很小，文字至少要占封面面积的1/4。用粗体、高对比色（白字红底、黄字黑底等），确保在小图状态下也能看清。
  2. 信息要少。封面最多放10个字以内的标题，不要堆砌信息。一句话戳中痛点就够了。
  3. 风格要统一。所有视频的封面保持统一的模板风格（字体、配色、排版），这样用户点进你的主页会觉得很整齐，更容易关注。

💬 四、评论区运营

评论区是最容易被忽视但价值极高的运营阵地。高互动率会让算法给你推更多流量：

  1. 发布后立刻自评。视频发布后的前30分钟是关键期。自己先评论一条引导性内容，比如"你们觉得哪个方法最有用？评论区告诉我"或者"需要完整教程的扣1"。这能带动其他用户参与评论。
  2. 及时回复每条评论。前1小时内尽量回复每一条评论。回复要真诚、有温度，不要只回复表情包。回复越长、越有内容，算法越认为这是高质量互动。
  3. 设置互动钩子。在视频结尾或文案中抛出一个问题，引导用户去评论区回答。比如"你是什么时候开始做短视频的？"这种低门槛问题，人人都能回答。
  4. 善用置顶评论。把最有价值或最能引发讨论的评论置顶。可以是自己的补充说明，也可以是粉丝的精彩评论。
